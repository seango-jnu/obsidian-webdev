
- 기본 셀렉터
  - class name
  - id name
  - tag name
  - 전체 셀렉터
    - `*`
    - Universal selector
  - 그룹 셀렉터
    - 콤마로 구분해서 여러 셀렉터에 동일 css 속성을 부여할 수 있음
- 속성 셀렉터
  - `[data-type='primary']`
  - `[data-type]`
  - `[href*='example.com']`
  - `[href^='https']`
  - `[href$='.com']`
