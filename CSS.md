- ## CSS 가 뭔가요?
	- CSS 는 사용자에게 문서를 표시하는 방법을 지정하는 언어입니다 — 스타일, 레이아웃 등.
	- 왜 필요한가요?
		- 문서의 매력과 가독성을 높입니다.
		- html 에 스타일을 표현하면 문서의 데이터적인 의미를 찾기 힘들어 집니다.
		- 키워드 : 시맨틱 마크업
